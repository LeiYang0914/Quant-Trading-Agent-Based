# Quant Alpha Research Project

A structured workspace for discovering, evaluating, and documenting quantitative alpha ideas in crypto and commodity markets.

## What This Project Is

- A research lab for alpha idea generation and documentation
- A source of structured handoff documents for a future Quant Programmer Agent
- A persistent memory system that tracks research state across sessions

## What This Project Is Not

- A trading system or backtesting framework
- A live trading bot
- Financial advice

## How to Use

1. Open this project in Claude Code
2. Claude reads `CLAUDE.md` → `PROJECT_STATE.md` → `ALPHA_BACKLOG.md` → `DECISIONS.md` automatically
3. Ask Claude to research an alpha idea, synthesize a paper, or explore a market inefficiency
4. Claude produces a research memo in `research_memos/` and/or a handoff doc in `handoff_to_programmer/pending/`
5. At session end, Claude updates the memory files

## Current Focus Areas

- Crypto perpetual futures: funding rate carry, crowding signals, open interest dynamics
- Commodity futures: term structure, carry, positioning-based signals
